## 🌟 Welcome, Future Contributor!

**FirstOpen** is a beginner-friendly repository designed to help you make your **very first Pull Request** to an open-source project. No experience required — just curiosity and a willingness to learn!

Whether you're fixing a typo, improving the UI, or adding a new feature, every contribution counts. This is *your* safe space to experiment, learn, and grow.

---

## 🎯 What You'll Learn

| Skill | What You'll Do |
|-------|----------------|
| 🍴 **Forking** | Create your own copy of a repo |
| 🌿 **Branching** | Work on features without breaking main |
| 💾 **Committing** | Save your changes with clear messages |
| 🚀 **Pushing** | Upload your branch to GitHub |
| 🔀 **Pull Requests** | Submit your changes for review |
| 🤝 **Collaboration** | Engage with maintainers & reviewers |
