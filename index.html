Your First PR in 5 Steps

From zero to merged — follow this interactive guide

1

Fork the Repository

Click the Fork button on GitHub to create your own copy of the project.

# Your forked repo will be at:
https://github.com/YOUR_USERNAME/first-open-contributions
2

Clone to Your Machine

Download the repository to your local development environment.

git clone https://github.com/YOUR_USERNAME/first-open-contributions.git
cd first-open-contributions
3

Create Your Branch

Always work on a new branch — never commit directly to main!

git checkout -b feature/my-first-contribution
4

Make Changes & Commit

Add your contribution, then stage and commit with a meaningful message.

# Make your changes to the files...
git add .
git commit -m "Add my first contribution: welcome page enhancement"
git push -u origin feature/my-first-contribution
5

Open Pull Request

Head back to GitHub, click "Compare & pull request", and submit your PR!

✓ Code reviewed ✓ Tests passing ✓ Merged to main
